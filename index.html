Concepto Clave: Los dos ejes

Para entender Flexbox, debes entender que existen dos ejes invisibles.

/* --------------------------------------
   EJE PRINCIPAL (Main Axis) ->
   Es la dirección en la que fluyen los ítems.
   (Por defecto: izquierda a derecha)
   -------------------------------------- */

/* |
   |  EJE CRUZADO (Cross Axis)
   |  Es perpendicular al eje principal.
   |  (Por defecto: arriba a abajo)
   v
*/
2 / 22

El Contenedor (Padre)

Todo empieza aplicando display: flex al elemento padre. Esto convierte a sus hijos directos en "flex items".

.contenedor {
  display: flex;
  
  /* Opcional: display: inline-flex;
     (El contenedor se comporta como inline,
      pero los hijos siguen siendo flex) */
}
3 / 22

1. Flex Direction (Padre)

Define cuál es el Eje Principal. ¡Esto cambia todo!

.contenedor {
  /* Horizontal (defecto) -> Eje Principal es X */
  flex-direction: row; 

  /* Vertical -> Eje Principal es Y */
  flex-direction: column;

  /* Inversos */
  flex-direction: row-reverse;
  flex-direction: column-reverse;
}
4 / 22

2. Flex Wrap (Padre)

Por defecto, Flexbox intenta meter todo en una sola línea (aplastando los ítems). Esto controla el salto de línea.

.contenedor {
  /* No saltar (defecto, puede causar scroll horizontal) */
  flex-wrap: nowrap;

  /* Si no cabe, baja a la siguiente línea */
  flex-wrap: wrap;

  /* Salta hacia arriba */
  flex-wrap: wrap-reverse;
}
5 / 22

3. Justify Content (Padre)

Alinea los ítems a lo largo del EJE PRINCIPAL (Main Axis). Controla la distribución horizontal (si es row).

.contenedor {
  justify-content: flex-start;    /* Inicio (izq) */
  justify-content: flex-end;      /* Final (der) */
  justify-content: center;        /* Centro */
  
  justify-content: space-between; /* Extremos pegados */
  justify-content: space-around;  /* Espacio alrededor */
  justify-content: space-evenly;  /* Espacio igual */
}
6 / 22

4. Align Items (Padre)

Alinea los ítems a lo largo del EJE CRUZADO (Cross Axis). Controla la alineación vertical (si es row).

.contenedor {
  align-items: stretch;     /* Estirar (defecto) */
  align-items: flex-start;  /* Arriba */
  align-items: flex-end;    /* Abajo */
  align-items: center;      /* Centro vertical */
  align-items: baseline;    /* Línea base del texto */
}
/* Nota: Si cambias a flex-direction: column,
   align-items controla la HORIZONTAL. */
7 / 22

5. Align Content (Padre)

Solo funciona si tienes flex-wrap: wrap y múltiples líneas. Controla el espacio entre las líneas enteras.

.contenedor {
  flex-wrap: wrap;
  height: 600px;
  
  /* Las líneas se agrupan al centro */
  align-content: center; 
  
  /* Las líneas se separan al máximo */
  align-content: space-between;
}
8 / 22

6. Gap (Padre)

La propiedad más amada. Crea espacio entre filas y columnas sin usar márgenes.

.contenedor {
  display: flex;
  
  /* Espacio en ambos ejes */
  gap: 20px;

  /* gap: fila columna; */
  gap: 10px 50px; 
}

Propiedades de los Hijos (Items)

Hasta aquí controlamos al padre. Pero los hijos también tienen propiedades individuales para controlar su tamaño y orden.

/* Ahora aplicaremos estilos a: */
.item {
   ...
}
10 / 22

7. Order (Hijo)

Cambia el orden visual sin alterar el HTML. Por defecto es 0.

.item-especial {
  /* Se va al final */
  order: 1; 
}

.item-primero {
  /* Se va al principio (incluso antes que 0) */
  order: -1;
}
11 / 22

8. Flex Grow (Hijo)

Define la capacidad de un ítem para crecer si hay espacio sobrante. Valor por defecto: 0 (no crecer).

.item {
  /* Todos crecen por igual para llenar el espacio */
  flex-grow: 1;
}

.item-vip {
  /* Crece el doble de rápido que los demás */
  flex-grow: 2;
}
12 / 22

9. Flex Shrink (Hijo)

Define la capacidad de un ítem para encogerse si falta espacio. Valor por defecto: 1 (sí encoger).

.logo {
  /* "No me encojas nunca, mantén mi tamaño" */
  flex-shrink: 0;
  width: 100px;
}

.texto {
  /* "Encógete más rápido si es necesario" */
  flex-shrink: 3;
}
13 / 22

10. Flex Basis (Hijo)

Es el tamaño inicial ideal del elemento antes de que se aplique grow o shrink. Reemplaza a width en el eje principal.

.item {
  flex-basis: 200px;
  /* El navegador intenta darle 200px.
     Luego mira si debe crecer o encoger. */
  
  flex-basis: auto; /* Mira su width/height */
}
14 / 22

11. Flex Shorthand (IMPORTANTE)

Se recomienda usar la propiedad corta flex en lugar de las 3 individuales. Orden: Grow | Shrink | Basis.

.item {
  /* flex-grow: 0, flex-shrink: 1, flex-basis: auto */
  flex: initial; /* (Defecto) */

  /* grow: 1, shrink: 1, basis: 0% */
  flex: 1; /* "Ocupa todo el espacio disponible" */

  /* grow: 0, shrink: 0, basis: auto */
  flex: none; /* Rígido */
}
15 / 22

12. Align Self (Hijo)

Permite a un hijo rebelarse contra el align-items del padre.

.contenedor {
  align-items: center;
}

.hijo-rebelde {
  /* Yo me voy arriba, aunque mi padre diga centro */
  align-self: flex-start;
}

Patrón: El Centrado Perfecto

El "Santo Grial" de CSS se resuelve en 3 líneas con Flexbox.

.centrado-total {
  display: flex;
  justify-content: center; /* Eje X */
  align-items: center;     /* Eje Y */
  
  height: 100vh;
}

Patrón: Navbar (Space Between)

Logo a la izquierda, Menú a la derecha.

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
}

/* 
   [Logo] <---------- espacio ----------> [Menú] 
*/
18 / 22

Patrón: Sticky Footer

Empujar el pie de página al fondo si hay poco contenido.

body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

main {
  /* Crece para ocupar todo el espacio libre */
  flex: 1; 
}
19 / 22

Truco: Margin Auto

En Flexbox, margin: auto se come todo el espacio disponible. Es mágico.

.item-derecha {
  /* Empuja al elemento totalmente a la derecha */
  margin-left: auto;
}
/* Útil en navbars para separar un grupo de iconos
   del resto de los elementos. */
20 / 22

Responsive Flexbox

Cambiar la dirección es la forma más fácil de hacer un diseño móvil.

.card {
  display: flex;
  flex-direction: row; /* Desktop: Lado a lado */
}

@media (max-width: 600px) {
  .card {
    flex-direction: column; /* Móvil: Uno encima de otro */
  }
}
21 / 22

Resumen Maestro

Recuerda siempre la dirección del eje.

1. Define 'display: flex' en el padre.
2. Decide 'flex-direction' (Row = X, Column = Y).
3. 'justify-content' mueve en el Eje Principal.
4. 'align-items' mueve en el Eje Cruzado.
5. Usa 'gap' para espacios.
6. Usa 'flex: 1' en hijos para llenar espacios.
